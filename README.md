# AfterBreach

An open-source, structured dataset and interactive tool for navigating cyber incident reporting obligations — starting with Australia.

- **Domain:** [afterbreach.org](https://afterbreach.org)
- **Status:** Scaffolding. Data is live; web app is not yet built.
- **License:** [Apache 2.0](LICENSE)

## What's here

- `data/` — the obligations dataset, one directory per jurisdiction.
- `data/schema.json` — JSON Schema defining the obligation shape.
- `web/` — interactive frontend (not yet built).
- `docs/` — contributor documentation.
